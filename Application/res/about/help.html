Third Eye

About App and User Manual

About app

Third eye is a video recording application. It was created as a part of the Final Year Project for a BS in Computer Science at the University of Central Asia.

The aim of the project and this application is development and implementation of real-time gaze tracking on a laptop screen as an additional method of proctoring online tests and examinations.

Learn more by visiting the GitHub repository for this project

About video recording

Third eye records your screen and web camera, arranges them according to the settings on the right side of the main window, and saves the recorded video on your disk.

Why does the app crash when I start recording?

The #1 reason for crashing is when the application cannot access your web camera.

If you are using an antivirus, try checking logs for untrusted applications. If this app is refused access to the web camera, add it as an exception or pause antivirus protection. Do not worry, this application is 100% safe, which you can verify by looking at the source code available on GitHub.

About gaze tracking

One of the main features of this app is gaze tracking which highlights the place on the screen at which the user is looking and detects when the user is not looking at the screen.

Unfortunately, accurate gaze tracking was not achieved during this project and its improvement is left to future contributions. The development process and implementation methods are documented in the FYP report, that can be used as a guide and reference.

Guide: buttons and their uses

Interface at launch

  1. Real-time feedback of your recorded video
  2. Recording status and control (start/stop)
  3. Gaze-tracking overlay settings - color
  4. Web camera settings - position on the recording
  5. Other settings and functions:
    • Change main window color theme (light/dark)
    • Open the folder where your recordings and logs are saved
    • Change the output folder to a different one
    • Start gaze tracking calibration
    • Close the application
Any changes you make are saved when you close the app so that on the next launch your app looks just as you have left it :)

Overlay color

This widget allows you to change the color of the gaze-tracking overlay. The rectangle shows the current color. The button opens a color picker in a pop-up window.

Webcamera position

This widget allows you to change the position of the web camera in the recorded video. Four buttons correspond to the the web camera position. The screenshots above show web camera in the bottom right and top left corners.

Themes

In the previous examples you saw the app window with Light theme applied. If you prefer darker colors, you can change the current theme by clicking the "Change theme" button.

Working with folders

The first screenshot demonstrates how "Open output folder" button works. The second image shows the pop-up window where you can select other folder to save your recordings.

Start/stop recording

Start and stop your recording by pressing the middle button below the main frame. Remember that the widgets on the right side of the app are disabled when the recording is running. To change any settings, stop the running recording first.
